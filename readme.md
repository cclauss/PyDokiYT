# PyDokiYT
PyDokiYT is a Python script to play Doki Doki on YouTube live!

## Installation
```
1. Download python. (if you don't have yet.)
2. Open cmd
3. Run pip install -r requirements.txt
4. Download main.py and clickme.bat
5. Run clickme.bat
```
